<h1 className="text-primary">Folder Setup</h1>

> In the pages folder **/pages/\_app.js|tsx**, **/pages/\_document.js|tsx** files. You are free to modify the meta tags and other content.

The folder structure should look like this:

```ts
bazar-react/
├── public/
|   └── assets/
|       └── images/
├── pages/
    ├── address
    ├── admin
    ├── orders
    ├── payment-methods
    ├── product
    ├── profile
    ├── shops
    ├── support
    ├── 50+ Other pages
├── src/
|   ├── animations
|   ├── components/
|   |   ├── layouts
|   |   |   ├── customer-dashboard
|   |   |   ├── vendor-dashboard
|   |   |   └── And shop layouts
|   |   └── Includes reusable atomic components
|   ├── contexts
|   ├── data
|   ├── fake-db
|   ├── hooks
|   ├── page-sections
|   |   └── Includes bigger components (Individual every pages sections compoents)
|   ├── theme
|   ├── utils
└── README.md
```
