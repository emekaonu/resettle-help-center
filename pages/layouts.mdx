<h1 className="text-primary">Layouts</h1>

The layouts are created inside the `src/components/layouts` folder. There are 6
different layouts for this project

1. `CustomerDashboardLayout`
2. `VendorDashboardLayout`
3. `ChcekoutNavLayout`
4. `SaleLayout`
5. `ShopLayout1`
6. `ShopLayout2`

<h4>1. CustomerDashboardLayout</h4>

```ts
// **Customer Dashboard Layout** used in below pages:

1. Wish List -> '/pages/wish-list'
2. Support Tickets -> '/pages/support-tickets'
3. Orders and Order Details Page -> '/pages/orders'
4. Profile and Edit Profile Page -> '/pages/profile'
5. Address and Address Details page -> '/pages/address'
6. Payment Methods and Payment Method Details Page -> '/pages/payment-methods'
```

<h4>2. VendorDashboardLayout</h4>

```ts
// **Vendor Dashboard Layout** used in below pages:

1. Inside vendor folder all pages -> '/pages/vendor'
2. Inside admin folder all pages -> '/pages/admin'
```
